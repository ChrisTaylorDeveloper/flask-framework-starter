## Getting Started

If not using Nix, create a virtual environment

```sh
python3 -m venv .venv
```

Then, activate the virtual environment

```sh
source .venv/bin/activate
```

Dependencies

```sh
pip install matplotlib
pip install PyQt6
```

```sh
pip install -r ./requirements.txt
```

### Prerequisites

* Python.
* Nix.

### Run Flask

```shell
flask --app finance_flask run
```
